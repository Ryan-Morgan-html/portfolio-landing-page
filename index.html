On the science end of things, one obviously has to be technically proficient enough for his code to function properly. On the other hand, even if the code is well written, the color scheme can still clash horribly, for example. Thus, for someone such as myself who comes from more of a generalist backgroud with a wide variety of interests (in ancient roman mosaics, for example), each step in the web development process holds a new challenge, and is thus all the more engaging.
